Concurrent Deviation Method

Concurrent Deviation Method correlation की direction (+/-) और उसकी approximate degree दोनों बताती है, लेकिन क्योंकि यह केवल signs (+/-) पर आधारित होती है और magnitude को ignore करती है, इसलिए इसे rough method कहा जाता है।

It considers only the direction of changes and ignores the magnitude of changes. Therefore, it provides only a rough measure of correlation.

Example

Year Sales (X) Profit (Y) X Change Y Change Same Direction?
2021 100 20 — — —
2022 120 25 + + Yes
2023 110 22 − − Yes
2024 130 28 + + Yes

तीनों बार दोनों variables एक ही दिशा में बदले।

इसलिए:

C = 3

(Concurrent Deviations = 3)

Formula

\[r = \pm \sqrt{\frac{2c - n}{n}}\]

Where

  • r = Coefficient of Correlation
  • C = Number of Concurrent Deviations
  • n = Number of Deviations

Sign of Correlation

Positive Sign (+)

  • C > n/2 → Positive Correlation (+)
n = 8
C = 6 
8/2 = 4
6 > 4 = (+)

Negative Sign

  • C < n/2 → Negative Correlation (-)
n = 8
C = 2
8/2 = 4
2 < 4 = (-)

Zero Sign

  • C = n/2 → लगभग No Correlation
n = 8
C = 4
8/2 = 4
4 == 4 = zero Correlation

Example

Data

Changes (+ / -)

Year Sales (X) Profit (Y) X Change Y Change Same Direction?
2020 100 50 — — —
2021 120 55 + + Yes
2022 110 52 − − Yes
2023 130 60 + + Yes
2024 125 58 − − Yes

सभी 4 बार changes एक ही दिशा में हुए।

C = 4

Concurrent Deviations = 4

Observations = 5

लेकिन deviations हमेशा एक कम होते हैं:

n = 5 - 1
n = 4

sign = (+)

4/2 = 2
4 > 2 = (+)
\[r = \pm \sqrt{\frac{2c - n}{n}}\]
\[r = + \sqrt{\frac{2 \times 4 - 4}{4}}\]
\[r = \frac{4}{4} = + 1\]

--> Perfect Positive Correlation

एक भी बार दोनों same direction में नहीं बदले।

C = 0

Total observations = 5

n = 5 - 1
n = 4

sign = (-)

4/2 = 2
0 < 2 = (-)

(जब majority changes opposite direction में हों, तब यही convenient form use की जाती है।)

\[r = \pm \sqrt{\frac{n - 2c}{n}}\]
\[r = - \sqrt{\frac{4 - 2 \times 0}{4}}\]
\[r = \sqrt{\frac{4}{4}} = - 1\]

--> Perfect Negative Correlation

Let's

n = 8
C = 6 
8/2 = 4
6 > 4 = (+)
\[r = \pm \sqrt{\frac{2C - n}{n}}\]
\[r = + \sqrt{\frac{2 \times 6 - 8}{8}}\]
\[\sqrt{\frac{4}{8}} = \sqrt{0.5} = + 0.71\]

--> Moderate Correlation

Let's

n = 8
C = 5 
8/2 = 4
5 > 4 = (+)
\[r = \pm \sqrt{\frac{2C - n}{n}}\]
\[r = + \sqrt{\frac{2 \times 5 - 8}{8}}\]
\[r = + \sqrt{\frac{2}{8}} = \sqrt{0.25} = 0.5\]

--> Moderate Correlation

Let's

n = 8
C = 1 
8/2 = 4
1 < 4 = (-)
\[r = \pm \sqrt{\frac{n - 2C}{n}}\]
\[r = - \sqrt{\frac{8 - 2 \times 1}{8}}\]
\[r = - \sqrt{\frac{6}{8}} = \sqrt{0.75} = -0.87\]

--> Hign Negative Correlation

अगर Karl Pearson's r अधिक accurate है, तो फिर यह method क्यों बनाई गई?

उत्तर: सुविधा और simplicity।

Karl Pearson's Method

Advantages

✔ बहुत accurate

✔ Correlation की exact degree बताती है

✔ Statistical analysis के लिए best

Problems

❌ Mean निकालना पड़ता है

❌ Deviations निकालने पड़ते हैं

❌ xy, x², y² निकालने पड़ते हैं

❌ बड़े data में calculation लंबी हो जाती है

Concurrent Deviation Method

Advantages

✔ केवल + और − signs देखने होते हैं

✔ बहुत जल्दी calculation हो जाती है

✔ Time Series Data में उपयोगी

✔ Rough idea तुरंत मिल जाता है

Problems

❌ Accuracy कम

❌ Magnitude को ignore करता है

❌ केवल approximate correlation देता है
