Quartile Deviation (Q.D)

1. Range

Meaning

  • Data की सबसे बड़ी और सबसे छोटी value के बीच का अंतर।

Need

  • Data का सबसे basic spread जानने के लिए।

Use Cases

  • Quick variability check
  • Quality control
  • Outliers का प्रारंभिक अंदाज़ा

Quick Intuition

"Data कितना फैला है (min से max तक)?"

Formula :

\[Range = L - S\]

L = Largest Value,
S = Samalles Value.

2. Coefficient of Range

Meaning

  • Range का relative (unit-free) measure।

Need

  • अलग-अलग units या datasets की Range compare करने के लिए।

Use Cases

  • Different datasets की variability compare करना।

Quick Intuition

"Range को comparison के लायक बनाता है।"

Formula :

\[Co-efficient\ of\ Range = \frac{L - S}{L + S}\]

Question

Find the range of weights of 7 students form the following.

27, 30, 35, 36, 38, 40, 43

\[Range = L - S\]
\[Range = 43 - 27\]
\[\boxed{Range = 16}\]
\[Co-efficient\ of\ Range = \frac{L - S}{L + S}\]
\[Co-efficient\ of\ Range = \frac{43 - 27}{43 + 27} = \frac{16}{70}\]
\[\boxed{Co-efficient\ of\ Range = 0.23}\]

3. Interquartile Range (IQR)

Meaning

  • \(Q_3\) और \(Q_1\)के बीच का spread।
  • बीच के 50% data का फैलाव।

Need

  • Outliers के प्रभाव से बचकर spread मापने के लिए।

Use Cases

  • Box Plot
  • Outlier Detection
  • Skewed data analysis

Quick Intuition

"Middle 50% data कितना फैला है?"

Formula :

\[Inter\ Quartile\ range = Q_3 - Q_1\]

4. Quartile Deviation (Semi-IQR)

Meaning

  • Interquartile Range का आधा।

Need

  • Middle 50% data की average spread जानने के लिए।

Use Cases

  • Skewed distributions
  • Outlier-resistant variability measure

Quick Intuition

"Middle 50% data का average spread।"

Formula :

\[Quartile\ Deviation = \frac{Q_3 - Q_1}{2}\]

5. Coefficient of Quartile Deviation

Meaning

  • Quartile Deviation का relative measure।

Need

  • अलग-अलग datasets की Quartile Deviation compare करने के लिए।

Use Cases

  • Relative variability comparison
  • Economics & Business datasets

Quick Intuition

"Quartile Deviation को comparison के लायक बनाता है।"

Formula :

\[Coefficient\ of\ Quartile\ Deviation = \frac{Q_3 - Q_1}{Q_3 + Q_1}\]

A. Quartile deviation - Individual Series

Example

Roll No. 1 2 3 4 5 6 7
Marks 20 28 40 30 50 60 52
Marks Ascending order 20 28 30 40 50 52 60
\[Q_1 = Size\ of\ \left( \frac{N+1}{4} \right)^{th} item\]
\[Q_1 = Size\ of\ \left( \frac{7+1}{4} \right)^{th} item\]
\[Q_1 = Size\ of\ \left( \frac{8}{4} \right)^{th} item\]
\[Q_1 = Size\ of\ \left( 2 \right)^{th} item\]
\[\boxed{Q_1 = 28}\]
\[Q_3 = Size\ of\ \left( 3\ \frac{N+1}{4} \right)^{th} item\]
\[Q_3 = Size\ of\ \left( 3\ \frac{7+1}{4} \right)^{th} item\]
\[Q_3 = Size\ of\ \left( 3\ \frac{8}{4} \right)^{th} item\]
\[Q_3 = Size\ of\ \left( 3\ \times 2 \right)^{th} item\]
\[Q_3 = Size\ of\ \left(6 \right)^{th} item\]
\[\boxed{Q_3 = 52}\]
\[Quartile\ Deviation = \frac{Q_3 - Q_1}{2}\]
\[Quartile\ Deviation = \frac{52 - 28}{2}\]
\[Quartile\ Deviation = \frac{24}{2}\]
\[\boxed{Quartile\ Deviation = 12}\]
\[Coefficient\ of\ Q.D = \frac{Q_3 - Q_1}{Q_3 + Q_1}\]
\[Coefficient\ of\ Q.D = \frac{52 - 28}{52 + 28}\]
\[Coefficient\ of\ Q.D = \frac{24}{80}\]
\[\boxed{Coefficient\ of\ Q.D = 0.3}\]
