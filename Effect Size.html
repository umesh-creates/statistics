Cohen's d (Effect Size) — Complete Example

Situation

एक company दो training programs compare कर रही है।

Program A Program B
Mean Score = 80 Mean Score = 90
SD = 20 SD = 20

मान लो t-test पहले ही बता चुका है:

p < 0.05

अर्थात statistically significant difference है।

अब सवाल:

Difference कितना बड़ा है?

Step 1: Formula

Formula :

\[d = \frac{\bar{X}_1 - \bar{X}_2}{Pooled SD}\]

Step 2: Values

\[\bar{X} = 80\]
\[\bar{X} = 90\]
\[Pooled SD = \sqrt{\frac{s_1^2 + s_2^2}{2}}\]
\[Pooled SD = \sqrt{\frac{20^2 + 20^2}{2}}\]
\[Pooled SD = \sqrt{\frac{400 + 400}{2}}\]
\[Pooled SD = \sqrt{\frac{800}{2}} = \sqrt{400} = 20\]
Pooled SD = 20

Step 3: Calculation

\[d = \frac{80 - 90}{20}\]
\[d = \frac{-10}{20} = -0.5\]

आमतौर पर magnitude देखते हैं:

\[|d| = 0.5\]

Rule of Thumb:

Cohen's d Meaning
0.2 Small
0.5 Medium
0.8+ Large

हमारे case में:

d = 0.5

Conclusion:

✅ Medium Effect

Final Report Style

t-test ने बताया:

p < 0.05

Cohen's d ने बताया:

d = 0.5

Final conclusion:

Programs statistically different हैं और difference का size medium है।

Notes Summary

p-value

बताती है:

Difference exist करता है या नहीं।

Confidence Interval

बताता है:

Parameter किस range में हो सकता है।

Effect Size

बताता है:

Difference कितना बड़ा है।
