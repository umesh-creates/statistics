Testing of Hypothesis

Errors

Type I Error (\(\alpha\) Error)

Type I Error = जब \(H_0\) सच हो, फिर भी हम उसे Reject कर दें।

👉 यानी सही चीज़ को गलत मान लेना।

Example

एक व्यक्ति वास्तव में निर्दोष (Innocent) है।

फिर भी court उसे दोषी (Guilty) घोषित कर देती है।

  • Reality: निर्दोष
  • Decision: दोषी

👉 यह Type I Error है।

Hypothesis Testing Language
  • \(H_0\) सच है
  • हमने \(H_0\) Reject कर दिया

👉 Type I Error (\(\alpha\) Error)

Trick

Type I Error = True H₀ को गलती से reject कर दिया । ✅

यही error \(\alpha\) (Level of Significance) द्वारा measure की जाती है।
इसलिए इसे \(\alpha\) Error भी कहते हैं। 🎯

Type II Error (\(\alpha\) Error)

Type II Error = जब \(H_0\) झूठा हो, फिर भी हम उसे Reject न करें।

👉 यानी गलत चीज़ को सही मान लेना।

Example

एक व्यक्ति वास्तव में दोषी (Guilty) है।

फिर भी court उसे निर्दोष (Innocent) छोड़ देती है।

  • Reality: दोषी
  • Decision: निर्दोष

👉 यह Type II Error है।

Hypothesis Testing Language
  • \(H_0\) झूठा है
  • हमने \(H_0\) Reject नहीं किया

👉 Type II Error (β Error)

Trick

Type II Error = False H₀ ko galti se Accept (Reject na karna) kar lena. ✅

याद रखने की Table

Reality Decision Error
\(H_0\) sach \(H_0\) Reject Type I Error (\(\alpha\))
\(H_0\) jhootha \(H_0\) Reject nahi Type II Error (\(\beta\))

Power of Test

Power of Test = किसी test की यह क्षमता कि वह झूठे \(H_0\) को सही तरीके से Reject कर सके।

👉 यानी जब \(H_0\) वास्तव में गलत हो, तो उसे पकड़ लेने की probability।

Formula

\(Power\ of\ Test = 1 − \beta\)

Example

मान लो:

  • \(\beta = 0.20 \ (20 \%)\)

तो

Power = 1 − 0.20 = 0.80 = 80%

👉 मतलब test के पास 80% chance है कि वह गलत \(H_0\) को पकड़कर Reject कर देगा।

Trick

  • \(\beta\) = गलत \(H_0\) को न पकड़ पाने की probability
  • Power = गलत \(H_0\) को पकड़ लेने की probability ✅

याद रखो:

High Power = Better Test 🎯

क्योंकि ऐसा test गलत \(H_0\) को अधिक बार सही तरीके से Reject करता है।
