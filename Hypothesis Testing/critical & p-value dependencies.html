Critical Value & p-value Dependencies

1. Z-Test

Critical Value depend करती है:

  • \(\alpha\) (Level of Significance)
  • Alternative Hypothesis
\[Critical\ Value = f(\alpha, H_1)\]

p-value depend करती है:

  • Calculated Z-score
  • Alternative Hypothesis
\[P-Value = f(Z, H_1)\]

2. t-Test

Critical Value depend करती है:

  • Level of Significance
  • Alternative Hypothesis
  • Degrees of Freedom (df)
\[Critical \ Value = f(\alpha, H_1, df)\]

p-value depend करती है:

  • Calculated t-score
  • Alternative Hypothesis
  • Degrees of Freedom (df)
\[P-Value = f(t, h_1, df)\]

3. Chi-Square Test

Critical Value Depends On

  • Level of Significance
  • degrees of freedom
\[Critical \ Value = f(\alpha, df)\]

p-value Depends On

  • Chi-Square Test
  • degrees of freedom
\[P-Value = f(\chi^2, df)\]

4. ANOVA

Critical Value Depends On

  • \(\alpha\)
  • \(df_B\)
  • \(df_W\)

p-value Depends On

  • \(F\ static\)
  • \(df_B\)
  • \(df_W\)
