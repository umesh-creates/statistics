Parametric Vs Non-Parametric

Parametric Test क्या होता है?

ऐसे tests जो कुछ assumptions मानकर चलते हैं।

जैसे

  • Data normal distribution follow करता है।
  • Mean meaningful है।
  • Variance आदि की assumptions पूरी होती हैं।

उदाहरण:

✅ Z-Test

✅ t-Test

✅ ANOVA

ये सभी Parametric Tests हैं।

Non-Parametric Test क्या होता है?

ऐसे tests जिन्हें कम assumptions चाहिए।

इनको कभी-कभी:

Distribution-Free Tests

भी कहते हैं।

ये तब use होते हैं जब:

  • Data normal नहीं है।
  • Outliers बहुत हैं।
  • Data ranks/ordinal form में है।
