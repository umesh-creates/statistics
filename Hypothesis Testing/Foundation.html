Testing of Hypothesis

Foundation

Population vs Sample

  • Population = पूरा group जिसके बारे में study करनी है। (जैसे: school के सभी 1000 students)
  • Sample = Population का छोटा हिस्सा जिस पर study की जाती है। (जैसे: उन्हीं 1000 में से 100 students)

Parameter vs Statistic

  • Parameter = Population का numerical measure। (जैसे: सभी students का actual average marks)
  • Statistic = Sample का numerical measure। (जैसे: चुने गए 100 students का average marks)

👉 Parameter → Population से आता है

👉 Statistic → Sample से आता है

Sampling Error

Sampling Error = Parameter और Statistic के बीच का difference।

Example

  • Population average marks = 70 (Parameter)
  • Sample average marks = 68 (Statistic)

Sampling Error = 70 − 68 = 2

👉 क्योंकि हमने पूरा Population नहीं लिया, केवल Sample लिया, इसलिए थोड़ा difference आ गया। यही Sampling Error कहलाता है। ✅

Null Hypothesis (H₀)

Null Hypothesis (H₀) = वह statement जिसे हम शुरू में सही मानकर चलते हैं।

👉 इसमें आमतौर पर "No Effect", "No Difference" या "No Change" माना जाता है।

Example

एक teacher कहता है:

Average marks = 70

So

\(H_0 : \mu = 70\)

अर्थात Population का average marks 70 है।

Hypothesis Testing में हमारा काम यह check करना होता है कि Sample data के आधार पर \(H_0\) को reject करना चाहिए या नहीं। ✅

Alternative Hypothesis (H₁ or Hₐ)

Alternative Hypothesis \((H_1)\) = वह statement जो Null Hypothesis \((H_0)\) के opposite होती है।

👉 यह बताती है कि Effect, Difference या Change मौजूद है।

If

\(H_0 : \mu = 70\)

तो Alternative Hypothesis हो सकती है:

  • \(H_1 : \mu \neq 70\) (Two-tailed)
  • \(H_1 : \mu > 70\) (Right-tailed)
  • \(H_1 : \mu < 70\) (Left-tailed)

अर्थात Population का average marks 70 नहीं है।

👉 \(H_0\) = No Difference
👉 \(H_1\) = Difference Exists ✅

Level of Significance \((\alpha)\)

Level of Significance \(\alpha\) = \(H_0\) को गलती से reject करने की maximum acceptable probability।

सरल शब्दों में:

👉 यह बताता है कि हम कितनी गलती का risk लेने को तैयार हैं।

Example

IF

\(\alpha\) = 0.05 (5%)

तो इसका मतलब है:

हम 5% chance तक \(H_0\) को गलत तरीके (गलती से) से reject करने का risk स्वीकार करते हैं।

Common Values

  • \(\alpha\) = 0.05 (5%) ✅ सबसे common
  • \(\alpha\) = 0.01 (1%)
  • \(\alpha\) = 0.10 (10%)

👉 \(\alpha\) जितना छोटा होगा, \(H_0\) को reject करना उतना कठिन होगा।

Confidence Level

Confidence Level = सही decision लेने का confidence।

यह Level of Significance \((\alpha)\) का opposite होता है।

IF

\(\alpha\) = 0.05% (5%)

Then

Confidence Level = 1 − 0.05 = 0.95 = 95%

👉 इसका मतलब है कि हमें अपने result पर 95% confidence है।

Trick

  • \(\alpha\) = Error ka Risk
  • Confidence Level = Sahi Hone ka Confidence ✅

p-value

p-value = यह बताती है कि अगर \(H_0 \ \) सच है, तो हमें मिला हुआ result कितना rare (असामान्य) है।

Simple Rule

  • p-value \(\leq \alpha → H_0 \ \) Reject ✅
  • p-value \(> \alpha → H_0\) Reject नहीं ❌

Example

मान लो:

  • \(\alpha\) = 0.05
  • p-value = 0.03

क्योंकि 0.03 < 0.05

👉 Result बहुत rare है, इसलिए \(H_0\) Reject कर देंगे।

Trick

छोटी p-value = \(H_0\) के खिलाफ strong evidence ✅

याद रखो:

p-value जितनी छोटी होगी, \(H_0\) को reject करने की संभावना उतनी अधिक होगी।

Critical Value

Critical Value = वह boundary (सीमा) value जिसके आधार पर हम decide करते हैं कि \(H_0\) reject करना है या नहीं।

👉 यह \(\alpha\) (Level of Significance) से निकाली जाती है।

Example

मान लो:

  • \(\alpha\) = 0.05
  • Two-tailed Z-test

तो Critical Values होंगी:

\(\pm\)1.96

  • Test Statistic −1.96 से कम या +1.96 से अधिक → H₀ Reject ✅
  • Test Statistic −1.96 और +1.96 के बीच → H₀ Reject नहीं ❌

Trick

👉 Critical Value = Decision Boundary (फैसला करने की सीमा). ✅

Rejection Region (Critical Region)

Rejection Region = वह area जहाँ Test Statistic पहुँच जाए, तो \(H_0\) Reject कर दिया जाता है।

👉 यह area Critical Value के बाहर होता है।

Example

यदि Critical Values:

\(\pm 1.96\)

हैं, तो

  • −1.96 से कम → Rejection Region
  • +1.96 से अधिक → Rejection Region

Trick

👉 Rejection Region = H₀ को Reject करने वाला क्षेत्र ✅

याद रखो:

Critical Value = Boundary (सीमा)

Rejection Region = Boundary के बाहर का area 🎯
