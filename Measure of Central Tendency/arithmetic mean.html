Measures of Central Tendency

Meaning

  • Data की central (typical) value बताने वाले measures।
  • पूरे dataset को एक single representative value में summarize करते हैं।

Need

  • Large data को जल्दी समझने के लिए।
  • Data का overall center जानने के लिए।
  • Different groups/data को compare करने के लिए।

Use Cases

  • Data summarization
  • Missing value imputation (Mean/Median/Mode)
  • Group comparison
  • Variance, Standard Deviation, Regression आदि के base के रूप में
  • Business decisions (average sales, income, marks, etc.)

Quick Intuition

"पूरे data को एक number में represent करना।"

Arithmetic Mean: Simple Arithmetic Average

\[\bar{X} = \frac{\sum X}{N}\]
\[\bar{X} = \frac{540}{10}\]

C. Continuous Series

Class Interval Frequency (f) Mid Points (m) fm
